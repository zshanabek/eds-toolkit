# EDS Extractor

Браузерный инструмент для работы с **Электронной Цифровой Подписью (ЭЦП)** Казахстана через локальное приложение **NCALayer**.

---

## Что такое ЭЦП?

**ЭЦП (Электронная Цифровая Подпись)** — это PKI-сертификат, выданный **Национальным Удостоверяющим Центром (НУЦ)** Республики Казахстан. Используется для:

- подтверждения личности по ИИН (Индивидуальный Идентификационный Номер)
- подписания документов и данных в электронном виде
- взаимодействия с государственными и корпоративными сервисами

---

## Возможности

### 1. Извлечение ЭЦП-сертификата

- Введите ИИН (12 цифр) — система проверяет его контрольную сумму
- Выберите файл ключа `.p12` через NCALayer
- Приложение подписывает аутентификационный пакет `{ action: "AUTHENTICATION", jwt_token: <ИИН> }`
- Результат отображается в двух форматах: **XML** и **Base64**

### 2. Подписание данных

- Введите произвольные данные (JSON, текст и др.)
- Приложение создаёт **CAdES-подпись** (CMS Advanced Electronic Signature)
- Результат — **Base64-строка**, готовая к отправке в backend
