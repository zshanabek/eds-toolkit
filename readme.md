# EDS Toolkit

Браузерный инструмент для работы с **Электронной Цифровой Подписью (ЭЦП)** Казахстана через [NCALayer](https://ncl.pki.gov.kz).

**Демо:** https://zshanabek.github.io/eds-toolkit/

---

## Для чего

Инструмент предназначен для **тестирования бэкенд-сервисов**, которые принимают ЭЦП-подпись. Типичный сценарий:

1. Вставьте JSON/текст в поле «Данные для подписания»
2. Подпишите через NCALayer — получите CMS в формате Base64
3. Скопируйте результат и вставьте в Postman (заголовок или тело запроса)
4. Отправьте запрос на эндпоинт, который валидирует ЭЦП

---

## Возможности

- **Аутентификационная подпись** — подписание аутентификационного пакета по ИИН, вывод результата в XML и Base64
- **Подписание данных** — CAdES-подпись (CMS) произвольных данных: JSON, текст, XML
- **Валидация ИИН** — проверка контрольной суммы по двухпроходному алгоритму НУЦ РК

## Стек

- Vanilla JS (без фреймворков)
- [ncalayer-js-client](https://github.com/sigex-kz/ncalayer-js-client) — WebSocket-клиент для NCALayer
- Tailwind CSS

## Как работает

```
Браузер → WebSocket (wss://127.0.0.1:13579) → NCALayer → .p12 файл / Казтокен / ID-карта
```

1. Пользователь вводит ИИН и выбирает файл ключа `.p12`
2. Приложение подключается к NCALayer по WebSocket
3. NCALayer открывает диалог выбора сертификата
4. Подписанный результат возвращается в XML и Base64

## Требования

- Установленный и запущенный [NCALayer](https://ncl.pki.gov.kz)
- Файл ключа ЭЦП (`.p12`) или аппаратный токен

## Запуск локально

```bash
git clone https://github.com/zshanabek/eds-toolkit.git
cd eds-toolkit
# Откройте index.html в браузере
```

> NCALayer должен быть запущен перед открытием страницы.

## Контекст

**ЭЦП (Электронная Цифровая Подпись)** — PKI-сертификат, выданный Национальным Удостоверяющим Центром (НУЦ) Республики Казахстан. Используется для подтверждения личности и подписания документов в государственных и корпоративных сервисах.

Поддерживаемые форматы подписи:
- **XAdES** — XML подпись (`signXml`)
- **CAdES** — подпись произвольных данных (`createCAdESFromBase64`)
